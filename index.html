Bookmarklets

lasercar/marklets

Bookmarklets (marklets) are JavaScript encased in a URL. Drag any them to your bookmarks bar to install it, then click it on any website to run it. Reloading the page will reset the script and clear the effects (unless it's designed to make persistant changes). Marklets are useful and powerful; because they run with the same permissions (client-side) as the website you run them on, it's good practice to be cautious about where you get your marklets from. I wrote all the marklets on this page, feel free to look at the source code of each if you feel the need.

    Developers
    To work properly, your javascript must be encased in a self-executing anonymous function (or void(0); at the end). It also needs to have semicolons in proper places, and have no single-line comments.

    (function() {
    your script here
    })();

    This website then encodeds the script to be URL-safe, and adds a javascript: prefix. The result is a bookmarklet!
